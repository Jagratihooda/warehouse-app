***Warehouse App***

***Tech Stack
Tools & Techniques used***

***Backend:***

Java 1.8
PostgreSQL
Spring Boot
Project Lombok
JUnit/Mockito

***Frontend:***

Angularjs
Bootstrap

***Tools:***

IntelliJ
Maven

***How to run***

Spring Boot will create tables.

Setup PostgreSQL with the configuration available in application.properties

Run with maven
mvn test
mvn spring-boot:run
Use app
Location
localhost:8081
